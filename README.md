# Image-to-Image Translation using pix2pix

This project demonstrates image-to-image translation using Stable Diffusion's pix2pix model.

## Features
- Converts input image into modified output
- Uses text-based instructions
- Based on conditional GAN (cGAN)

## Technologies Used
- Python
- PyTorch
- Diffusers
- Google Colab

## Example
Input: Original Image  
Output: Painting Style Image
